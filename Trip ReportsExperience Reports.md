# Trip Reports/Experience Reports 
#### Guidelines: 

*   [Combination Section](#root/HKR7Wkjdt69j/NCBt6ij6pcg4)
*   A substance that which is naturally occurring as one body but contains a set of constituents (ex. Alkaloids, terpenoids etc) is still considered as one substance 
*   Psychoactive and non-psychoactive substances are considered and to be explored 
*   ‘Reports’ to be considered as one of the following: “Trip" “Experience” “Overdose”
*   Exempt combinations are included but color coded (See [Combination Section](#root/HKR7Wkjdt69j/NCBt6ij6pcg4) for more details)
*   Current Total - 117 at Zopiclone
